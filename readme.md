### 源码地址

项目从 github地址: [https://github.com/tinypuppet/nxdc-milktea](https://github.com/tinypuppet/nxdc-milktea) fork进行开发学习

1. 本项目包含：

	- 首页
	- 点餐（自取和外卖两种方式，有基本的点餐逻辑处理）
	- 取餐
	- 我的
	- 积分商城
	- 积分商城详情页
	- 积分签到
	- 会员码
	- 我的卡券
	- 收货地址
	- 我的资料
	- 我的订单
	- 订单详情
	- 余额
	- 会员卷包
	- 卷包详情
	- 订单评价
	- 买茶送包
	- 更多服务
	- 开发票
	- 礼品卡

2. 所有数据都是静态的，保存在api文件夹下。
